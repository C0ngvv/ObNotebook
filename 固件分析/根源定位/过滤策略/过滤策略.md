### strcpy
```c
char *strcpy(char *dest, const char *src)
```

#### src
- src是否可控，若为常量字符串则不可控
#### dest
- dest是否为全局变量，无法栈溢出
- dest的缓冲区长度

### strncpy, memcpy
```c
char *strncpy(char *dest, const char *src, int length)
```

#### src
- src是否可控，若为常量字符串则不可控
#### dest
- dest是否为全局变量，无法栈溢出
- dest的缓冲区长度
#### length
- 长度为数字，且大于dest缓冲区长度，skip
